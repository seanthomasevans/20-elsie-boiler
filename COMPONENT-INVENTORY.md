# Component Inventory - 20 Elsie #217 Boiler Replacement

## What You HAVE (From Existing System)

### 1. Grundfos Circulation Pump
- **Photos**: IMG_0456, IMG_0470, IMG_0471
- **Model**: Likely UP15-42 or UP15-58 (black cast-iron body, copper sweat connections)
- **Status**: Reusable if it still spins freely. Power it briefly to test. These last 15-20 years.
- **Use in new system**: Heating loop circulation pump

### 2. Precision Plumbing Products Flow Control Valve
- **Photos**: IMG_0465, IMG_0466, IMG_0467, IMG_0468, IMG_0469
- **Description**: Brass inline "Trane Valve", rated 25-75 PSI, with flow direction arrow
- **Manufacturer**: Precision Plumbing Products, Portland OR
- **Status**: Reusable as a balancing valve on the return line, but may not be needed since the Rinnai self-modulates

### 3. Gas Shutoff Valve + CSST Line
- **Photos**: IMG_0472, IMG_0473, IMG_0474, IMG_0475
- **Description**: 3/4" yellow CSST (corrugated stainless steel tubing) with brass compression fittings and ball valve (yellow handle)
- **Status**: Reusable. Reroute the terminal end to the Rinnai's gas port. CSST fittings reusable if undamaged.

### 4. Air Handler with Control Board
- **Photos**: IMG_0440, IMG_0457, IMG_0458
- **Brand**: Energy Saving (Lennox/Aire-Flo OEM)
- **Control board terminals**: Freeze stat, heating mode, fan speed (low/med/high), zone thermostat, 24V transformer input, motor
- **Status**: Stays. The AH is the fan coil. It doesn't care what heats the water.

### 5. 24V Relay / Terminal Block
- **Photos**: IMG_0444
- **Description**: Small red terminal block on wall with BX cable. Part of control wiring between thermostat, transformer, and zone valve.
- **Status**: Reusable as-is.

### 6. Copper Piping + CPVC Adapters
- **Photos**: IMG_0444, IMG_0459
- **Description**: Existing soldered copper runs with CPVC transition fittings
- **Status**: Some runs reusable if in the right location. SharkBite push-connect fits directly onto cleaned copper.

### 7. Wilkins Aqua-Gard Vacuum Breaker
- **Photos**: IMG_0482, IMG_0483
- **Manufacturer**: Zurn/Wilkins
- **Description**: Backflow preventer cap for potable water protection
- **Status**: Reusable. Important for open-loop heating config. Code may require it.

### 8. Pipe Size Confirmation
- **Photos**: IMG_0476, IMG_0477, IMG_0478, IMG_0479
- **Measurement**: ~19mm OD on Helios vernier caliper = 3/4" copper nominal (7/8" OD)
- **Result**: SharkBite 3/4" fittings will fit directly
