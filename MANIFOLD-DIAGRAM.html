FLOW LOGIC

1Cold supply enters from main shutoff
2Passes BV1, feeds expansion tank, enters unit
3Cold also branches to TMV cold bypass
4Hot exits unit through BV2
5Tee splits: right to TMV, left to heating
6Heating supply passes ZV + BV3 to fan coil
7TMV mixes hot+cold to 120F for house DHW
8Heating return from fan coil (via existing circ pump) through BV4 + check valve back to cold line
MAKING IT BEAUTIFUL:

1. All four vertical drops from unit at exactly 2" spacing, plumb and parallel
2. All ball valves at the SAME height (40" from floor)
3. Use copper for everything within 24" of unit, transition to PEX for long runs
4. P-clamp pipes to wall with consistent spacing
5. SharkBite transitions at PEX-to-copper joints (red caps for hot, blue for cold)
6. Expansion tanks either both floor-mounted or both wall-hung at same height
7. Label every valve with adhesive tags
8. Run pipes parallel, never crossing unless unavoidable
9. Reference: your neighbor's Rinnai Empower install (photo 579) is the standard
